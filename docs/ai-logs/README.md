# AIログ（public）

Copilot とのやりとりを、公開向けに **マスキング/整形** したログを置きます。

## Sessions
会話内容_1〜3 は、意図的に **それぞれ別のチャットセッション（独立したコンテキスト）** として実施しています。
同一セッションの続きをそのまま積み上げるのではなく、
**製造（実装）** と **レビュー（要件・設計・テスト観点の確認）** を分けることで、前提の引きずりを減らし、より客観的に見直しやすくする狙いがあります。

- [会話内容_1.md](public/会話内容_1.md): 設計・実装のレビュー
- [会話内容_2.md](public/会話内容_2.md): テスト観点の洗い出し〜テスト実装
- [会話内容_3.md](public/会話内容_3.md): 責務分離の改善（リファクタリング）と資料整合

## Policy
- 個人情報/社内情報/環境依存情報（ユーザー名、PCパス、`file:///` リンク等）は削除または一般化
- Copilot の内部ログ（"Created todos" / "Starting" / "Made changes" 等）は原則削除

## Regenerate
```bash
python scripts/sanitize_ai_logs.py
```
